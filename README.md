# 📊 Sales Data Analysis using Python

## 🚀 Project Overview

This project performs end-to-end analysis on a sales dataset using Python. It includes data cleaning, exploratory data analysis (EDA), visualization, statistical testing, and machine learning models to extract meaningful insights.

---

## 📁 Dataset

The dataset contains sales records with features like:

* Order Date
* Region
* Item Type
* Units Sold
* Total Cost
* Total Revenue
* Total Profit

---

## 🔧 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SciPy
* Scikit-learn

---

## 📊 Key Features

### 🔹 Data Preprocessing

* Converted date columns into datetime format
* Extracted Year and Month
* Handled missing values
* Created Profit Margin feature

### 🔹 Data Visualization

* Sales by Region (Bar Plot)
* Profit Distribution (Histogram)
* Monthly Sales Trend (Line Chart)
* Correlation Heatmap
* Revenue Share (Pie Chart)
* Cost vs Revenue (Scatter Plot)
* Item Type Distribution

### 🔹 Statistical Analysis

* T-Test (Profit comparison)
* Shapiro Test (Normality check)
* Chi-Square Test (Categorical relationship)

### 🔹 Machine Learning Models

* Linear Regression (Revenue Prediction)
* Logistic Regression (Profit Classification)

---

## 📈 Results

* Identified trends in monthly sales
* Found correlation between cost, revenue, and profit
* Built predictive models with good accuracy
